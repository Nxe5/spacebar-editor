# VS Code Icons (default file icon pack)

Spacebar Editor ships the **Icons – Maintained** icon set as the default explorer file/folder theme.

Icons are used under the MIT License. This directory contains `manifest.json` (from the extension’s `icons.json`) and SVG assets from the `icons/` folder.

To refresh from upstream:

```bash
pnpm run sync-icons
```

Or use **Settings → Appearance → Refresh default icon pack** in the desktop app (downloads the latest marketplace VSIX).
